== Installation ==

You can install the Business Badges plugin from your WordPress Dashboard or manually via FTP. For more details, visit http://qualisure.es

= From WordPress Dashboard =

1. Navigate to 'Plugins -> Add New' from your WordPress dashboard.
2. Search for `Business Badges` and install it.
3. Activate the plugin from Plugins menu.
4. Add any widget to your widget zone from Appearance -> Widgets and configure the widget options.

= Manual Installation =

1. Download the plugin file: `business-badges.zip`
2. Unzip the file
3. Upload the`business-badges` folder to your `/wp-content/plugins` directory (do not rename the folder)
4. Activate the plugin from Plugins menu.
5. Add any widget to your widget zone from 'Appearance -> Widgets' and configure the widget options.
